0  1  2  3
sort-n  sort-f  sort-c  sort-s

Написать 4 функции, сортирующие массив 4 способами: методом пузырька, вставки, выбора и быстрой сортировкой. 
0 вариант - массив целых чисел
1 вариант - массив вещественных чисел
2 вариант - массив символов
3 вариант - массив строк

Эти функции поместить в отдельный файл, сделать к нему заголовочный файл. 

Программа должна запускаться из командной строки:

0: sort [-s|-b|-i|-q] [-2] <имя файла> (по умолчанию -q)
1: sort [-s|-b|-i|-q] [-f|-e|-g] <имя файла> (по умолчанию -q -g)
2: sort [-s|-b|-i|-q] [-t|-2] <имя файла> (по умолчанию -q -t)
3: sort [-s|-b|-i|-q] <имя файла> (по умолчанию -q)

-s - сортировка методом выбора
-b - методом пузырька
-i - методом вставки
-q - быстрая сортировка
(по умолчанию -q)

-2 - числа читаются и записываются в двоичной системе

-f, -e, -g - форматы записи вещественных чисел

-t, -2 - открытие файла как текстового или как бинарного

Числа, символы или строки читаются из указанного файла, сортируются указанным способом и выводятся на экран.
